При запуске файла test_ETD.py, предлагается ввести поисковой запрос
По нему формируются две переменные:
request_text - поисковой запрос для гугла
folder_name - название папки, куда будут сохраняться картинки
Создается, папка.
Осуществляется поиск в гугл-картинках
Загружаются, первые 15, которые можно скачать и сохраняются в папке с названием содержащимся в folder_name в той же директории.
Завершение программы.

Сторонние библиотеки:
pip install icrawler
